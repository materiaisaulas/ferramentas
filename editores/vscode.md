## O que é o Visual Studio Code

O **Visual Studio Code (VSCode)** é um editor de código-fonte desenvolvido pela Microsoft, amplamente utilizado no desenvolvimento de software. Ele foi projetado para ser **leve, extensível e multiplataforma**, funcionando em sistemas operacionais como Windows, Linux e macOS.

Diferentemente de ambientes de desenvolvimento mais pesados (IDEs completas), o VSCode adota uma abordagem modular: sua funcionalidade pode ser ampliada por meio de **extensões**, permitindo que o editor seja adaptado a diferentes linguagens de programação e tipos de projeto.

Entre suas principais características estão:

* edição de código com destaque de sintaxe
* autocompletar inteligente (IntelliSense)
* integração com sistemas de controle de versão (Git)
* terminal integrado
* suporte a depuração de programas
* grande ecossistema de extensões

Essas características tornam o VSCode uma ferramenta adequada tanto para **estudantes iniciantes** quanto para **desenvolvedores profissionais**.

### Estrutura geral do VSCode

A interface do VSCode é composta por alguns elementos principais.

**Barra lateral (Activity Bar)**
Localizada à esquerda da tela, permite acessar rapidamente funcionalidades como:

* explorador de arquivos
* controle de versão (Git)
* execução e depuração
* extensões instaladas

**Explorador de arquivos (Explorer)**
Permite visualizar e organizar a estrutura de diretórios do projeto.

**Editor de código**
Área central onde os arquivos são abertos e editados.

**Terminal integrado**
Permite executar comandos diretamente dentro do editor, como execução de programas, instalação de bibliotecas ou comandos do Git.

**Painel de extensões**
Permite instalar funcionalidades adicionais que ampliam as capacidades do editor.

---

### Extensões importantes

Uma das principais vantagens do VSCode é seu **sistema de extensões**. Essas extensões adicionam suporte a linguagens, ferramentas de depuração e integração com diversos frameworks.

Algumas extensões amplamente utilizadas em cursos de programação são:

**Python**

Suporte completo para desenvolvimento em Python, incluindo:

* execução de scripts
* depuração
* sugestões de código

**Flutter**

Permite desenvolver aplicações móveis utilizando Flutter e Dart diretamente no VSCode.

**GitLens**

Amplia as funcionalidades do Git, permitindo visualizar histórico de alterações e autoria de linhas de código.

**Prettier**

Ferramenta para formatação automática de código, mantendo padrão de estilo.

---

### Execução de programas

O VSCode permite executar programas diretamente pelo **terminal integrado**.

Exemplo de execução de um programa Python:

```bash
python programa.py
```

Também é possível configurar **botões de execução** e ambientes de depuração para facilitar o desenvolvimento.
